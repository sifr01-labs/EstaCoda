# التشغيل الأول

هذه الصفحة ستكون النسخة العربية لمراجعة الصفحة الإنجليزية المرتبطة.

## الغرض

ابدأ EstaCoda وتابع باستخدام معالج التهيئة.

## القنوات الاختيارية

يمكن لـ onboarding الأول إعداد قنوات اختيارية. عند اختيار ⁨WhatsApp⁩، تشغل ⁨EstaCoda⁩ تدفق إعداد ⁨QR⁩ نفسه المستخدم في ⁨Setup Editor⁩ والأمر ⁨estacoda whatsapp⁩: يفحص حزمة الجسر المعزولة، ويسأل قبل إصلاح الاعتمادات، ويعرض رمز ⁨QR⁩ في الطرفية، ولا يكتب إعداد ⁨WhatsApp⁩ أو حالة الجلسة إلا بعد نجاح الاقتران.

إذا رُفض إصلاح الاعتمادات أو فشل، أو انتهت مهلة ⁨QR⁩ أو فشل الاقتران، يسجل onboarding حالة ⁨WhatsApp⁩ كتخطي أو إعداد غير مكتمل ويستمر. لا تُكتب إعدادات ⁨WhatsApp⁩ جزئية في هذه المسارات.

## ملاحظات مهمة

- اسماء الأوامر، الاختيارات، والمعاملات التقنية تبقى باللغة الإنجليزية.
- مسارات الملفات، ومتغيرات البيئة، ومعرفات المزودين تبقى باللغة الإنجليزية.
- لا تزال الترجمة الآلية الكاملة في هذه المرحلة.

## مصدر الحقيقة

مجموعة وثائق Docusaurus هذه هي المصدر العام لمطالبات v0.1.0 الموجهة للمستخدم. إذا خالفت السلوك المنفذ، يجب إصلاح الوثائق أو الكود قبل الشحن.
