# التشغيل الأول

## الغرض

## ملاحظات مهمة

- اسماء الأوامر، الاختيارات، والمعاملات التقنية تبقى باللغة الإنجليزية.
- مسارات الملفات، ومتغيرات البيئة، ومعرفات المزودين تبقى باللغة الإنجليزية.
- لا تزال الترجمة الآلية الكاملة في هذه المرحلة.
