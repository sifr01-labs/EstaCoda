# Stage Guide

| Stage | Output artifact |
|---|---|
| Idea | Product Definition (problem, user, use case, success condition, non-goals) |
| Research | Research Map (facts, patterns, references, open questions, unknowns) |
| Engine | Engine Map (input, processing, storage, output, dependencies, failure points, approval points) |
| Decision gate | Decision Log Entry or Decision Gate (options, downstream impact, reversibility) |
| PRD | Phased PRD (phased goals, acceptance criteria, tests, risks; Phase 1 = vertical slice) |
| Builder agent | Builder Agent Brief (inspection rules, vertical slice, small commits, validation, decision gates) |
| Documentation | Control artifacts (architecture map, dependency map, README, roadmap, risk register, decision log) |
| Reviewer agent | Reviewer Agent Briefs (architecture, security, product, testing distinct scopes, evidence required) |
| Review comparison | Review Comparison (consensus, divergence, evidence needed, recommended default) |
| Builder response | Builder Response Request (accept/reject/modify per recommendation with rationale) |
| Milestone review | Milestone Review (what works, what's fragile, debt created, next vertical slice) |
| Product control | `PRODUCT_CONTROL.md` (thesis, architecture, roadmap, decisions, risks, debt, next milestone) |
